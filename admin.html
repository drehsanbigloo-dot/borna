تصاویر سایت

مدیریت تمام تصاویر و ویدیوهای استفاده شده در سایت. می‌توانید فایل جدید آپلود کرده یا مسیر فایل را تغییر دهید.

خروجی و پشتیبان‌گیری

تمامی داده‌های سایت را به صورت یک فایل JSON ذخیره کنید یا از فایل پشتیبان بازیابی کنید.

خروجی گرفتن (Export JSON)

تمام محتوای ویرایش شده را به صورت یک فایل JSON دانلود کنید. این فایل شامل تمام برگه‌ها و تنظیمات است.

بازیابی از فایل (Import JSON)

فایل JSON پشتیبان را انتخاب کنید تا تمام داده‌های سایت با اطلاعات فایل جایگزین شود.

بازنشانی به مقادیر پیش‌فرض

تمامی تغییرات را پاک کرده و به محتوای پیش‌فرض اولیه بازنشانی کنید.
